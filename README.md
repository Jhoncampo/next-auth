## Información

- Clonar el repositorio
- Instalar las dependencias "npm install" 
- Inciar prisma "npx prisma"
- En el archivo ".env" colocar los datos de la base de datos
- Crear el/los modelos de las tablas en schema.prisma
- hacer la migración "npx prisma migrate dev --name init" el "--name init" es opcional
